# Job Notification Tracker

A simple job notification tracking application with KodNest Premium Design System.

## Features
- Dashboard, Saved, Digest, Settings, and Proof pages
- Responsive navigation with mobile hamburger menu
- Off-white background (#F7F6F3)
- Deep red accent (#8B0000)
- Serif headings with high whitespace layout

## How to Run
Simply open `index.html` in your web browser.
